# Slate / Black Theme Brief

## Target Mood

The site should feel like:
- slate
- black steel
- matte glass
- terminal-light glow
- modern portfolio, not gaming UI

The result should be dark and confident, but not neon-heavy and not pure black everywhere.

## Visual System

### Color

Recommended palette roles:
- `canvas`: near-black with blue/slate bias
- `surface`: charcoal slate
- `surfaceStrong`: graphite
- `border`: cool gray with low contrast
- `textPrimary`: soft white
- `textSecondary`: muted stone/slate
- `accent`: ice blue, steel cyan, or desaturated teal

Avoid:
- saturated purple gradients as the default background
- bright rainbow accents
- flat pure-black everywhere with no depth

### Typography

Direction:
- One strong display face for the hero and section headings
- One highly readable sans for body copy and UI

Use type to carry personality, not illustration overload.

### Surfaces

Prefer:
- layered dark panels
- subtle borders
- restrained glow on interactive states
- soft spotlight or grid textures in the background

Avoid:
- busy SVG overlays as the primary section identity
- different background gimmicks for every section

### Motion

Prefer:
- staggered fade/slide reveals
- gentle parallax or spotlight movement if added carefully
- hover states that reinforce hierarchy

Avoid:
- constant floating animation on multiple large assets
- motion that competes with readability

## Section Notes

### Home

Use the hero to establish:
- name
- role
- short value proposition
- primary CTA
- supporting social links

The hero should carry the brand with typography, spacing, and lighting, not cartoon imagery.

### About

Present:
- short bio
- location
- specialties
- approach to product and engineering

This section should feel editorial and credible.

### Projects

Shift from generic description to concrete examples.
Even if some work is private, show:
- selected public repos
- categories of work
- technologies
- what kind of problems you solve

### Contact

Keep the form clean, low-friction, and trustworthy.
Dark inputs should still have strong focus states and legible helper text.

## Asset Direction

Likely keep:
- logo if it still works on dark backgrounds
- resume assets
- profile image if recolored/cropped well

Likely replace or retire:
- bright decorative section illustrations that do not fit the new tone
- section-specific shiny SVG overlays

## Success Test

The redesign is successful if the homepage looks intentional even before a visitor reads the copy.
